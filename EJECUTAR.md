EN LINUX

CREAR EL ENTORNO VIRTUAL

python3 -m venv .venv

ACTIVAR EL ENTORNO VIRTUAL

source .venv/bin/activate

INSTALAR LAS DEPENDENCIAS

pip install -r requirements.txt

EJECUTAR

python app.py

para salir de modo virtual
deactivate
